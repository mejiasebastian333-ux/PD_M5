# Emausoft Analytics Project

## Descripción
Proyecto de análisis de datos end-to-end para Emausoft, integrando ventas, clientes, productos y regiones para generar insights de negocio.

## Estructura del Proyecto
- `contexto_pd/`: Documentos de contexto de la prueba de desempeño.
- `data/`: Datos crudos, procesados e integrados.
- `docs/`: Documentación de cada etapa.
- `outputs/`: Reportes y exportaciones finales.
- `scripts/`: Código Python organizado por etapas.
- `sql/`: Esquemas y consultas SQL.
- `assets/`: Recursos visuales como diagramas.

## Instalación
1. Clona el repositorio.
2. Instala dependencias: `pip install -r requirements.txt`
3. Ejecuta el pipeline: `python scripts/pipeline.py`

## Uso
Sigue el flujo definido en `contexto_pd/flujoPD.md`.
